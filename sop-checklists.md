# HAMZURY — Standard Operating Procedure (SOP) Checklists
# Generated from company documents — for review before system build

## 1. HAMZURY CSO (Client Services & Operations)
*Service: Client intake, onboarding, relationship management*

### Pre-Task
- [ ] Inquiry received and logged in Central Master Tracker
- [ ] Client brief document requested and received
- [ ] Scope of work confirmed with client
- [ ] Service package selected and confirmed
- [ ] Proposal sent to client
- [ ] Proposal approved and signed
- [ ] Invoice raised and payment confirmed
- [ ] Project reference number (CLT-XXX) assigned
- [ ] Relevant department(s) briefed

### During Task
- [ ] Kickoff call/meeting held with client
- [ ] Project timeline communicated to client
- [ ] Weekly check-in scheduled
- [ ] Client questions and concerns logged
- [ ] Progress update sent to client (mid-project)
- [ ] Any scope changes documented and approved

### Post-Task
- [ ] Final deliverable reviewed and approved internally
- [ ] Deliverable sent to client
- [ ] Client acknowledgement received
- [ ] Client feedback collected
- [ ] Final invoice raised (if applicable)
- [ ] Project status updated to "Delivered" in tracker
- [ ] Client record archived
- [ ] Referral or testimonial requested
